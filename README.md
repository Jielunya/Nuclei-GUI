一款基于Python开发的Nuclei图形化工具，提供直观的模板管理和批量扫描功能，使用前先将本工具放到Nuclei同级目录或配置全局环境变量。

## 功能特性

### 🎯 核心功能
| 功能模块 | 描述 |
|---------|------|
| **模板管理** | 官方POC模板自动加载与更新，自定义POC模板支持 |
| **目标管理** | 单目标/批量目标扫描支持，目标列表可视化管理 |
| **扫描引擎** | 基于Nuclei引擎，支持多种协议和扫描模式 |
| **结果输出** | 彩色终端输出，结果自动保存，实时扫描状态显示 |

### ⚡ 技术特性
- **多线程扫描**: 非阻塞UI设计，扫描任务后台运行
- **智能缓存**: 模板列表自动缓存，提升启动速度
- **ANSI色彩支持**: 终端输出彩色高亮，提升可读性
- **代理支持**: HTTP/HTTPS/等代理配置
- **自定义模板**: 支持递归加载文件夹中的YAML模板

### 🔧 实用功能
- 模板搜索和过滤
- 批量操作（全选/取消全选）
- 扫描进度实时显示
